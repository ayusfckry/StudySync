# StudySync - Project Requirements Document

## 📌 Overview
StudySync is a web-based platform that combines a **study planner** with a **study buddy matching system**. It helps students plan their study schedules and collaborate with peers.

## 🎯 Objectives
- Improve time management
- Enable collaborative learning
- Increase productivity

## 👥 Target Users
- Students (school/college)

## 🧩 Core Features
- User Authentication
- Study Planner
- Study Buddy Matching
- Study Rooms
- Progress Tracking

## ⚙️ Functional Requirements
- Create/manage study plans
- Match with study buddies
- Track progress

## 🔒 Non-Functional Requirements
- Responsive UI
- Fast performance
- Secure login
